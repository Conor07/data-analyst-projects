# Data Analyst Portfolio

## SQL Project:

Investigating job data, specifically looking for insights into Data Analyst and Dats Scientist jobs.

### Creating Databases:

The .csv files that need to be used in the [sql_load](/sql_load/) directory to create the database locally:
https://drive.google.com/drive/folders/1moeWYoUtUklJO6NJdWo9OV8zWjRn0rjN
Then you should be able to run the .sql files in [sql_project](/sql_project/).

### What I learned:

-- 1. The importance of data cleaning and preparation before analysis, as seen in the initial steps of creating tables and loading data.
-- 2. The value of joining multiple tables to gain comprehensive insights, such as combining job postings with skills data to analyze demand and salary trends.
-- 3. SQL skills: writing complex queries, using CTEs for better readability and performance, and applying aggregate functions to derive meaningful insights from the data.

### Data Analysis Conclusions:

I ultimately learned that skills such as 'pytorch' and 'tensorflow' are among the top-paying skills, however, critical analyst skills such as 'python' and 'sql' also show high average salaries, indicating that both technical and analytical skills are valuable in the job market for these roles. This suggests that while specialized skills can lead to higher salaries, foundational skills remain essential for career growth in the data field.
